# Calculator

## Project Description
The Calculator is a simple web-based application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and helps in understanding JavaScript functions, event handling, and DOM manipulation.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear / Erase functionality
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User clicks the number buttons to enter values
2. User selects an arithmetic operation
3. User clicks the equals button to get the result
4. Clear button is used to erase the input or reset the calculator

## Project Type
Learning Project (Internship Task)

## Learning Outcome
- Understanding JavaScript functions
- Practice with event handling
- Improved logic building
- Hands-on experience with DOM manipulation
